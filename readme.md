# Projeto modelo

Projeto modelo para criacao de novos projetos.

Contem
- Laravel 5.3
- Laravel Collective
- Laravel 5 Repositories
- Tradução das mensagens de erros para Português do Brasil

# Instalação

1 - clone o repositório

2 - cd LaravelNewProject

3 - composer install

4 - php artisan vendor:publish
